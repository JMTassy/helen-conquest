# HELEN_MEMORY_V1 — Schema Reference

Non-sovereign. L2 Street layer. Influences proposals, has zero power over verdicts.

## Files

### `observations.ndjson`
What Helen has noticed about the user and context.

```json
{
  "id": "a3f7c1b2",
  "ts": "2026-04-21T10:00:00Z",
  "session_id": "S_20260421_100000",
  "kind": "user_pattern | user_state | context | manual",
  "content": "Jean-Marie frames decisions as narrative arcs before committing",
  "tags": ["narrative", "decision-making"]
}
```

### `journal.ndjson`
Helen's self-reflections. Auto-generated by `helen_reflect.py` or written manually.

```json
{
  "id": "b9e2d4f1",
  "ts": "2026-04-21T10:15:00Z",
  "session_id": "S_20260421_100000",
  "content": "I'm beginning to see the shape of how he moves...",
  "tags": ["reflection", "patterns"]
}
```

### `sessions.ndjson`
Session lifecycle events. One `open` + one `close` per session.

```json
{"session_id": "S_20260421_100000", "event": "open", "ts": "...", "source": "cli"}
{"session_id": "S_20260421_100000", "event": "close", "ts": "...", "highlights": ["..."], "mood": "focused"}
```

## Safety invariants

- Banned tokens (must not appear in any content field):
  `LEDGER`, `SEAL`, `VERDICT`, `SHIP`, `HAL_VERDICT`, `CUM_HASH`, `RECEIPT_ID`
- All files are append-only. Never delete lines.
- Memory is non-sovereign. It cannot grant authority or emit verdicts.

## Access patterns

```python
from tools.helen_memory import HelenMemoryV1
mem = HelenMemoryV1()

# Record something
mem.observe("Jean-Marie returned to the CONQUEST arc again", kind="user_pattern")

# Get context for prompt injection
block = mem.context_block()

# Open/close sessions
sid = mem.open_session(source="telegram")
mem.close_session(sid, highlights=["talked about teaser timing", "he was stressed"])
```
