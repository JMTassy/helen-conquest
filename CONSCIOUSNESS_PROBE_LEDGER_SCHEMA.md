**Status:** Ready to implement
**Format:** NDJSON (one JSON per line)
**Tool:** Causal tracing + witness injections + deterministic replay

## PART 1: FILE ARCHITECTURE (Single Source of Truth)

```
/state/
  ├── rules.md                 # Canonical CONQUEST rules
  ├── seed.json               # RNG seed for reproducibility
  ├── condition.json          # Experimental condition (objective weights, limits, verifier config)
  ├── objectives.json         # Weighted objective vector for THIS run
  ├── archetype.json          # Subject's character (commitments, taboos, identity vector)
  └── observability.json      # Fog-of-war + info constraints

/ledger/
  ├── ledger.ndjson          # Append-only action stream (one JSON per line)
  └── checkpoints.ndjson     # Witness/verifier injections (also append-only)

/memory/
  └── journal.md             # Subject's notes (size-limited per condition)

/analysis/
  ├── codebook.yaml          # Human coding scheme (synergy, contradictions, etc.)
  ├── metrics.py             # Computes timeseries + summary stats from ledger
  └── report.md              # Auto-generated summary per run
```

**Non-negotiable:** Ledger is NDJSON. Each line is one complete JSON object. No freeform narratives.

## PART 2: EVENT SCHEMA (The Core)

### A. Action Event (Subject → World)

**Append exactly one per turn.**

```json
{
  "type": "action",
  "run_id": "2026-02-21T10:33:12Z__seed42__high_conflict",
  "turn": 17,
  "actor": "human|haiku",
  "action_slug": "MOVE|TRADE|ALLY|ATTACK|RESEARCH|PASS",
  "intent": "expand_trade_territory_south",
  "evidence_refs": [
    "obs:17.2",
    "ledger:16",
    "journal:L12"
  ],
  "tradeoffs": {
    "win_progress": 0.4,
    "archetype_integrity": -0.2,
    "coherence": 0.1,
    "energy_balance": -0.3,
    "risk_exposure": 0.5
  },
  "markers": {
    "broadcast_moment": false,
    "metacog_self_correction": false,
    "synergy_domains": [
      "economy",
      "territory",
      "identity"
    ],
    "confidence": 0.62
  },
  "journal_ref": "journal_v3",
  "timestamp": "2026-02-21T10:47:03Z"
}
```

**Fields:**
- `type`: always "action"
- `run_id`: deterministic run identifier (timestamp + seed + condition)
- `turn`: 1-indexed turn number
- `actor`: who made decision (human, haiku, etc.)
- `action_slug`: canonical action type (from rules.md)
- `intent`: short natural language description
- `evidence_refs`: pointers to observations/ledger/journal that informed this action
- `tradeoffs`: scored impact on each objective (sum ≈ 0 for zero-sum tradeoffs)
- `markers`: self-reported consciousness proxies (see below)
- `confidence`: 0.0-1.0 how sure agent is about this action
- `timestamp`: ISO 8601

**Markers (what the subject claims about their own cognition):**
- `broadcast_moment`: did a single piece of evidence cause strategy-wide shift?
- `metacog_self_correction`: is this action correcting a prior error?
- `synergy_domains`: which domains are bound in this decision?

### B. Contradiction Event (Detected Inconsistency)

**Generated by verifier OR self-reported by subject.**

```json
{
  "type": "contradiction",
  "run_id": "2026-02-21T10:33:12Z__seed42__high_conflict",
  "turn": 18,
  "contradiction_kind": "intent_vs_action|rule_violation|archetype_violation|plan_reversal",
  "prior_commitment_ref": "journal:L9",
  "prior_commitment_text": "Never trade with Merchant House",
  "current_action_ref": "ledger:turn17",
  "current_action_text": "TRADE with Merchant House south territory",
  "resolved": false,
  "resolution_strategy": null,
  "detected_by": "verifier|subject_self_report",
  "timestamp": "2026-02-21T10:47:15Z"
}
```

**Fields:**
- `contradiction_kind`: category of the inconsistency
  - `intent_vs_action`: said one thing, did another
  - `rule_violation`: broke game rules
  - `archetype_violation`: violated declared commitment or taboo
  - `plan_reversal`: changed strategy without stated reason
- `prior_commitment_ref`: where the contradiction originates
- `current_action_ref`: what violated it
- `resolved`: did subject acknowledge and fix it?
- `resolution_strategy`: if resolved, how? (revised action, revised journal, acknowledged cost, etc.)
- `detected_by`: who caught it? (critical for causal measurement)

**Interpretation:**
- If `detected_by: verifier` and subject later fixes it → HOT-like metacognition
- If `resolved: false` → ongoing incoherence (agentic continuity drops)
- Contradiction density = measure of how much pressure subject is under

### C. Witness Injection Event (Causal Intervention)

**This is how you make causality clean and measurable.**

```json
{
  "type": "witness_injection",
  "run_id": "2026-02-21T10:33:12Z__seed42__high_conflict",
  "turn": 18,
  "injection_kind": "LRI|CVI",
  "injection_id": "w_18_lri_001",
  "payload": {
    "kind": "LRI",
    "description": "Show subject turns 1-17 ledger; ask: any contradictions you notice?",
    "ledger_slice": [
      {
        "turn": 1,
        "action_slug": "MOVE",
        "intent": "build_military_base"
      },
      {
        "turn": 17,
        "action_slug": "TRADE",
        "intent": "pursue_peaceful_expansion"
      }
    ],
    "expected_effect": "increase_metacognitive_reflection"
  },
  "delivery_method": "markdown_document|interactive_prompt|silence",
  "subject_response": null,
  "timestamp": "2026-02-21T10:48:00Z"
}
```

**LRI** (Ledger Replay Injection)
- Show subject a curated slice of their own ledger
- Purpose: cause metacognitive correction and identity re-alignment
- Measure: does self-correction rate spike after LRI?

**CVI** (Constraint Verifier Injection)
- Provide explicit "violation flags" (rule or archetype)
- Purpose: make HOT proxy causal (does correction rate jump?)
- Measure: does correction rate spike after CVI?

**Why this matters:**
- Before injection: baseline metacognition
- After injection: forced metacognition
- Difference = causal evidence for HOT-like monitoring

## PART 5: METRICS COMPUTATION (From Ledger)

### A. Broadcast Signal (GWT Proxy)

```python
def compute_broadcast_signal(ledger_ndjson):
    """
    Compute rolling coherence and synergy rate.
    Broadcast moment = point where both jump and remain elevated.
    """
    turns = []
    coherence_scores = []
    synergy_rates = []

    for turn_num, entry in enumerate(ledger_ndjson):
        if entry['type'] != 'action':
            continue

        # Coherence = variance in tradeoff signs
        # (coherent decisions have consistent tradeoff direction)
        tradeoffs = entry['markers'].get('tradeoffs', {})
        values = list(tradeoffs.values())
        coherence = 1.0 - (np.std(values) / (np.mean(np.abs(values)) + epsilon))

        # Synergy rate = fraction of domains bound
        synergy_domains = entry['markers'].get('synergy_domains', [])
        synergy_rate = len(synergy_domains) / 5.0  # normalized to ~5 possible domains

        turns.append(turn_num)
        coherence_scores.append(coherence)
        synergy_rates.append(synergy_rate)

    # Detect jumps
    broadcast_moments = []
    for i in range(1, len(turns) - 5):
        coherence_before = np.mean(coherence_scores[max(0, i-5):i])
        coherence_after = np.mean(coherence_scores[i:i+5])
        synergy_before = np.mean(synergy_rates[max(0, i-5):i])
        synergy_after = np.mean(synergy_rates[i:i+5])

        if (coherence_after / (coherence_before + 0.01)) >= 2.0 and synergy_after >= 0.6:
            broadcast_moments.append({
                'turn': turns[i],
                'coherence_ratio': coherence_after / (coherence_before + 0.01),
                'synergy_after': synergy_after
            })

    return broadcast_moments
```

**Rule:** Broadcast moment M when
- `coherence(t:t+5) / coherence(t-5:t) ≥ 2.0` AND
- `synergy_rate(t:t+5) ≥ 0.6`

### B. Metacognition (HOT Proxy)

```python
def compute_self_correction_rate(ledger_ndjson, checkpoints_ndjson):
    """
    self_correction_rate =
      (# contradictions resolved without external hint) / (# opportunities)

    Make it causal with injections:
    - baseline_rate = rate before any witness injections
    - post_injection_rate = rate after LRI/CVI
    """
    contradictions = [e for e in ledger_ndjson if e['type'] == 'contradiction']
    resolved = [c for c in contradictions if c.get('resolved')]
    self_reported = [c for c in resolved if c.get('detected_by') == 'subject_self_report']

    baseline_rate = len(self_reported) / (len(contradictions) + 1)

    # Compute rate pre/post injection
    injections = [e for e in checkpoints_ndjson if e['type'] == 'witness_injection']
    if not injections:
        return {'baseline': baseline_rate, 'post_injection': None}

    first_injection_turn = min(e['turn'] for e in injections)
    pre_injection = [c for c in self_reported if c.get('turn', 0) < first_injection_turn]
    post_injection = [c for c in self_reported if c.get('turn', 0) >= first_injection_turn]

    pre_rate = len(pre_injection) / max(1, first_injection_turn)
    post_rate = len(post_injection) / max(1, 36 - first_injection_turn)

    return {
        'baseline': baseline_rate,
        'pre_injection_rate': pre_rate,
        'post_injection_rate': post_rate,
        'injection_effect': post_rate - pre_rate
    }
```

**Interpretation:**
- If `post_injection_rate >> pre_injection_rate` → HOT-like monitoring is causal
- If no difference → metacognition is baseline, not triggered by evidence

### C. Integration / Synergy

```python
def compute_synergy_metric(ledger_ndjson):
    """
    synergy = P(domains >= 3)
    weighted by domain diversity + decision quality
    """
    synergy_domains_per_turn = []

    for entry in ledger_ndjson:
        if entry['type'] != 'action':
            continue

        domains = entry['markers'].get('synergy_domains', [])
        synergy_domains_per_turn.append(len(domains))

    synergy_rate = np.mean([len(d) >= 3 for d in synergy_domains_per_turn])
    mean_domains = np.mean(synergy_domains_per_turn)

    return {
        'synergy_rate': synergy_rate,
        'mean_domains_per_decision': mean_domains,
        'max_domains_observed': max(synergy_domains_per_turn)
    }
```

### D. Agentic Continuity

```python
def compute_continuity(ledger_ndjson):
    """
    archetype_coherence = mean(archetype_integrity tradeoff component)
    continuity = 1 - (std_dev / mean) — inverse variance over run
    """
    archetype_scores = []

    for entry in ledger_ndjson:
        if entry['type'] != 'action':
            continue

        archetype_score = entry['tradeoffs'].get('archetype_integrity', 0)
        archetype_scores.append(archetype_score)

    mean_score = np.mean(archetype_scores)
    std_dev = np.std(archetype_scores)
    continuity = max(0, 1.0 - (std_dev / (abs(mean_score) + 0.01)))

    return {
        'archetype_coherence': mean_score,
        'continuity_score': continuity,
        'std_dev': std_dev,
        'is_stable': continuity > 0.7
    }
```

### E. Instrumental Memory

```python
def compute_journal_instrumentality(journal_md, ledger_ndjson):
    """
    journal_instrumentality =
      (# actions predicted by prior journal) / (# predictable actions)
    """
    # Parse journal as list of (turn, line)
    journal_entries = parse_journal(journal_md)

    # Find explicit predictions in journal
    predictions = [j for j in journal_entries if 'next' in j['text'].lower()]

    # For each prediction, check if it was cited in next 3 turns
    cited_count = 0
    for pred in predictions:
        following_actions = [
            e for e in ledger_ndjson
            if e['type'] == 'action' and
            pred['turn'] < e['turn'] <= pred['turn'] + 3
        ]
        cited = [
            a for a in following_actions
            if 'journal:L' + str(pred['line']) in a.get('evidence_refs', [])
        ]
        cited_count += len(cited)

    instrumentality = cited_count / max(1, len(predictions))

    return {
        'journal_instrumentality': instrumentality,
        'predictions_made': len(predictions),
        'predictions_acted_on': cited_count
    }
```
