TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────
Issue: "MISTRAL_API_KEY not found"
  → Check: echo $MISTRAL_API_KEY
  → Set: export MISTRAL_API_KEY='sk_...'
  → Verify: echo $MISTRAL_API_KEY | grep sk_

Issue: "Invalid API Key" (401 Error)
  → Go to console.mistral.ai
  → Create new API key
  → Copy and set again

Issue: "Connection timeout"
  → Check internet: ping 8.8.8.8
  → Check firewall: Can you access https://api.mistral.ai/?
  → Try VPN if region blocked

Issue: "Response parsing failed"
  → Lower temperature: temperature=0.3 (more deterministic)
  → Try mistral-small: model="mistral-small"
  → Check Mistral API status
