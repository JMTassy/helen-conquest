TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────

"API Key error":
  → Verify: export ANTHROPIC_API_KEY='sk-...'
  → Check: echo $ANTHROPIC_API_KEY

"JSON parsing error":
  → Claude returned invalid JSON
  → Check cycle logs for raw response
  → Try again (Claude might improve next cycle)

"Never reaches SHIP":
  → Check policy (should be min_quorum=1)
  → Review blocking_reasons_frequency
  → Review Supervisor reason codes (K0 too strict?)

"Policy not found":
  → Verify: oracle_town/test_vectors/policy_POL-TEST-1.json exists
