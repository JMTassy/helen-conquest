THE 3-SKILL SYSTEM (Same everywhere)
═════════════════════════════════════════════════════════════════════════════════

FetchSkill → fetch(url)
  └─ Replace: requests.get(url)
  └─ Kernel decides: Approve/deny based on domain policy
  └─ Logs: URL, status code, content length

ShellSkill → execute(command)
  └─ Replace: subprocess.run(command)
  └─ Kernel decides: Approve/deny based on command policy
  └─ Logs: Command, output, exit code

MemorySkill → write(key, value) / read(key)
  └─ Replace: self.data[key] = value
  └─ Kernel decides: Approve/deny writes (reads are free)
  └─ Logs: Key, value size, operation type
