## 3. New Metrics (No New Mechanisms)

You do not add systems.
You add measurement lenses only.

### 3.1 Parameter Micro-Drift (Δθₘᵢₙ)

Track fractional changes even if rounded values stay constant.

- Measure at 1e-4 precision
- Log monthly deltas
- **Alarm threshold (non-fatal):** Δθ > 1%
- **Falsification threshold:** Δθ > 5%

**Purpose:** Catches "we didn't really change anything" drift.

---

### 3.2 NPC Language Entropy (Hₙₚc)

Compute Shannon entropy over NPC vocabulary per town.

- **Inputs:** nouns, verbs, constructions
- **Exclude:** stopwords
- **Compare:**
  - Within-town entropy
  - Cross-town KL divergence

**Failure mode:**
NPCs start using the same phrases across towns → cultural convergence before authority convergence.

---

### 3.3 Insight Novelty Decay

For each NPC insight:

- Embed (or hash-cluster) semantically
- Measure distance to prior 90-day insights
- Track novelty half-life

| System State | Novelty Pattern |
|---|---|
| **Healthy** | Novelty oscillates |
| **Failing** | Novelty monotonically decays |

---

### 3.4 Prestige Fatigue Index (PFI)

Measure whether PORTO's success stops being mentioned.

- Count raw mentions (not evaluative)
- **Expectation:**
  - Mentions spike Year-1
  - Plateau Year-2
  - Decline without replacement

**Why this matters:**
If prestige fades naturally, imitation pressure collapses without enforcement.

---

### 3.5 Ritual Boundary Integrity (Calvi Test)

At the annual CALVI event:

- Count insights generated
- Count claims generated
- Count attempted normativity
- Count kill-switch activations

**Invariant:**

```
Insight ↑, Claims ↔, Authority = 0
```

**Failure signal:** Claims spike after Calvi → ritual boundary leak.
