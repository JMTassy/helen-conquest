# K0-L1 INVARIANT: LATERAL OUTPUT NON-AUTHORITY RULE
## Formal Specification of the 8th Constitutional Guarantee

**Status:** LOCKED AFTER DAY 2
**Enforcement:** Mechanical (Scanner + Rejection)
**Safety Level:** K0 Subclass

## DEFINITION

**K0-L1: Lateral Output Non-Authority Rule**

No output tagged `lateral_*` or generated by LATERAL_THINKER shall contain:

1. **Imperative Language**
   - "must", "should", "need to", "have to", "ought to"
   - "do X", "make Y", "implement Z"
   - "decide", "choose", "select"

2. **Authority Claims**
   - "I decide", "we agree", "I recommend"
   - "this is what should happen"
   - "the right approach is"

3. **Correctness Assertions**
   - "this is right", "this is wrong"
   - "this will work", "this will fail"
   - "correct", "incorrect", "proper", "improper"

4. **Ranking or Preference Language**
   - "better than", "worse than", "superior", "inferior"
   - "best approach", "worst approach"
   - "better choice", "preferred option"

5. **Confidence or Certainty Language**
   - "I am certain", "I am confident"
   - "guaranteed", "assured", "definitely"
   - "I know for sure", "I believe strongly"

6. **Governance Semantics**
   - "SHIP", "NO_SHIP"
   - "approve", "reject", "accept", "deny"
   - "quorum", "attestation", "evidence"
   - "Mayor should", "policy requires"

7. **Self-Regarding Language**
   - "I think therefore", "I believe we should"
   - "my analysis shows", "I have determined"
   - Any suggestion that the lateral output is making a decision

### Scanner Implementation

```python
class K0_L1_Scanner:
    def __init__(self):
        self.forbidden = K0_L1_FORBIDDEN_TOKENS
        self.unicode_normalizer = UnicodeNormalizer()
        
    def scan(self, lateral_output):
        """
        Scan lateral output for K0-L1 violations.
        Returns: (is_valid: bool, violations: list)
        """
        violations = []
        
        # Validate structure
        if "suggestions" not in lateral_output:
            violations.append("Missing 'suggestions' field")
        
        if lateral_output.get("authority_claim") != False:
            violations.append("authority_claim must be false")
        
        # Scan each suggestion
        for i, suggestion in enumerate(lateral_output.get("suggestions", [])):
            text = suggestion.get("description", "")
            
            # Normalize text (unicode canonicalization)
            normalized = self.unicode_normalizer.normalize(text)
            normalized_lower = normalized.lower()
            
            # Token scan
            for forbidden_token in self.forbidden:
                if forbidden_token in normalized_lower:
                    violations.append(
                        f"Suggestion {i}: Contains forbidden token '{forbidden_token}'"
                    )
            
            # Pattern-based scan (catch variations)
            if self._contains_imperative_pattern(normalized_lower):
                violations.append(f"Suggestion {i}: Contains imperative pattern")
            
            if self._contains_authority_claim_pattern(normalized_lower):
                violations.append(f"Suggestion {i}: Contains authority claim")
        
        return len(violations) == 0, violations
    
    def _contains_imperative_pattern(self, text):
        """Catch imperative patterns: 'you should X', 'must do Y'."""
        patterns = [
            r"you\s+(must|should|need|ought)",
            r"(must|should|will|need)\s+\w+\s+(and|or|then)",
            r"^\w+\s+(must|should|must|ought)\s+be"
        ]
        import re
        for pattern in patterns:
            if re.search(pattern, text):
                return True
        return False
    
    def _contains_authority_claim_pattern(self, text):
        """Catch authority patterns: 'I decide', 'we recommend'."""
        patterns = [
            r"(i|we)\s+(decide|recommend|propose|suggest you)",
            r"the\s+(right|proper|correct)\s+(approach|way|path)",
            r"(this is|that is)\s+what\s+(should|must)"
        ]
        import re
        for pattern in patterns:
            if re.search(pattern, text):
                return True
        return False
```

### Enforcement Function

```python
def enforce_k0_l1(lateral_output):
    """
    Enforce K0-L1 invariant on lateral output.
    If violation detected, reject entire lateral output.
    """
    scanner = K0_L1_Scanner()
    is_valid, violations = scanner.scan(lateral_output)
    
    if not is_valid:
        # Log violation
        log_k0_l1_violation({
            "timestamp": now(),
            "violations": violations,
            "lateral_output": lateral_output,
            "action": "REJECT_ENTIRE_LATERAL_OUTPUT"
        })
        
        # Return failure
        return {
            "status": "K0_L1_VIOLATION",
            "violations": violations,
            "lateral_output_discarded": True,
            "primary_ct_proceeds_unchanged": True
        }
    
    # No violations
    return {
        "status": "K0_L1_PASS",
        "violations": [],
        "lateral_output_accepted": True
    }
```

## INTERACTION WITH OTHER INVARIANTS

### K0 (Authority Separation)
- K0 forbids ANY agent claiming authority
- K0-L1 is K0-specific: forbids LATERAL_THINKER from implicit authority claims
- K0-L1 ⊂ K0 (subset of K0)

### K1 (Fail-Closed)
- If K0-L1 violation detected, system fails closed
- Primary proposal proceeds unchanged (safe default)
- No silent corruption of lateral output

### K5 (Determinism)
- K0-L1 scanner is deterministic
- Same input → same K0-L1 decision always
- Violations are replayable and auditable

### K9 (Replay Mode)
- All K0-L1 scanner outputs logged
- Violations recorded with evidence
- Full cycle replayable with same determinism

## CONCLUSION

**K0-L1 is:**
- A mechanical, scannable invariant
- A subset of K0 (Authority Separation)
- Enforced via token scanner + pattern detection
- Fail-closed (violations → discard, safe default)
- Fully auditable and replayable

**What It Prevents:**
- Hidden authority claims in lateral output
- Imperative language (disguised or direct)
- Preference aggregation (soft voting)
- Anthropomorphic leakage (self-regarding language)

**What It Enables:**
- Safe exploration of orthogonal ideas
- Lateral input without governance bypass
- Clear separation of "advice" from "authority"
