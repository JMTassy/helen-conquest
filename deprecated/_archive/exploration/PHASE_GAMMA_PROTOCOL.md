# PHASE γ: SCALE (Second Cohort Protocol)
**Testing Civilization Coherence at 10x Agent Population**

**Protocol Date**: 2026-03-04 (Post-Week 3 Assessment)
**Objective**: Add second cohort of 10 new agents; observe if signal maintains or degrades at scale
**Duration**: 3 weeks (parallel to existing 5-agent cohort, or sequential)
**Success Threshold**: 8/10+ coherence maintained; no signal corruption
**Failure Threshold**: <7/10 coherence; breakdown of constitutional system

---

## DECISION: STRUCTURE

### Option A: Parallel (Simultaneous)
- Keep original 5 agents active
- Add 10 new agents in separate realm
- After 1 week, merge realms
- Observe mixed-cohort dynamics

**Advantage**: Controlled merge, can isolate issues
**Disadvantage**: More complex to track

### Option B: Sequential (Clean Slate)
- Original 5 agents conclude Week 3
- Assessment period (1 week)
- New 10 agents start fresh Week 1
- Follow same 3-week cycle
- Compare results to original cohort

**Advantage**: Cleanest data, easier assessment
**Disadvantage**: Longer timeline, no live merge dynamics

**Recommendation**: **Option B (Sequential)**

Rationale: Clean data > merged dynamics. Merge can happen in Phase δ.

### Week γ2: CONSTRAINT (Days 8-14)

**Constraint**: 2 artifacts max per district (same as original Week 2)

**New element**: What happens when original 5 and new 10 operate under same constraint?

**Observation focus**:

| Metric | Track | Meaning |
|--------|-------|---------|
| Quality improvement | New cohort Week 1→2 vs original cohort Week 1→2 | Does constraint improve quality uniformly? |
| Collaboration under pressure | Cross-cohort references increase/stay/decrease | Does constraint force original + new to work together? |
| Conflict emergence | Do new agents challenge original constitution? | Does pressure expose disagreement? |
| Role stability | Do new agent roles merge with or diverge from original 5? | Is 10-agent system still 5-role, or has it expanded? |

**Critical test**:

If new cohort reaches same constitutional design as original cohort independently, that validates signal strength.

If new cohort proposes different governance, that reveals alternatives.

Either way, you learn something about constitutional uniqueness.

**Output goals**:
- 20 new artifacts (constraint reduces output)
- 20+ cross-cohort references (pressure should force collaboration)
- 1-3 proposed amendments from new cohort (challenge via pressure)

**Pass condition**:
- New cohort quality improves under constraint (same pattern as original Week 2)
- Mixed cohort (old + new) can collaborate without friction
- Amendments are coherent (not contradictory)

## ASSESSMENT FRAMEWORK (Post-Week γ3)

### Assessment Template

Use same structure as **CONQUEST_TOWN_WEEK3_ASSESSMENT.md**, but with comparative analysis:

```
PART 1: Quantitative Comparison
- Compare new cohort metrics to original cohort
- Expected: Similar trajectory (8/10+ coherence, 40+ artifacts Week 1, etc.)
- Variance: <15% = success, >30% = concerning

PART 2: Constitutional Coherence
- New cohort's governance vs original cohort's governance
- Compare thematic alignment (both landed on same principles?)
- Compare structural soundness (both solutions equally robust?)

PART 3: Cross-Cohort Dynamics
- Original + new agents working together (mixed teams)
- Are they compatible? Friction? Synergy?
- Can one person from new cohort join original cohort's district?

PART 4: Emergence Patterns
- Did new cohort independently invent same mechanisms?
- Did new cohort invent DIFFERENT but compatible mechanisms?
- Did new cohort discover vulnerabilities in original design?

PART 5: Fatigue Analysis
- New cohort energy across Week 1-3
- Any degradation? Any burnout? Any over-engagement?

PART 6: Verdict
- PASS (8/10+ coherence, compatible governance, scale-ready)
- CONDITIONAL (7-8/10 coherence, minor incompatibilities, needs refinement)
- FAIL (<7/10 coherence, governance incompatibility, needs redesign)
```

### Pass/Fail Decision Matrix

| Metric | Pass | Conditional | Fail |
|--------|------|-----------|------|
| Coherence (new cohort) | 8-10/10 | 7-8/10 | <7/10 |
| Compatibility (old + new) | No friction | Minor friction | Major conflict |
| Constitutional uniqueness | Same or compatible | Different but compatible | Incompatible |
| Energy trajectory | Sustained | Slight fatigue | Burnout |
| Artifacts output | 40+ Week 1, 20 Week 2, 40+ Week 3 | 30-40 Week 1, 15 Week 2, 30 Week 3 | <30 any week |

**Overall Pass**: 4+ metrics in PASS column
**Overall Conditional**: 2-3 metrics in PASS, 2+ in CONDITIONAL
**Overall Fail**: 2+ metrics in FAIL column
