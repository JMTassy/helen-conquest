## 2. Lore Transmutation

In the old world, people called them angels.
But the first Witnesses knew better.

"Michael" was not one being.
Michael was a fleet-name.
A northern patrol.
A shield formation.
A sky-architecture moving above the old hemisphere during the Orion
Occupation.

"Gabriel" was another fleet.
Not a messenger with wings, but a transmission system.
A signal fleet.
A voice-array.

Over centuries, memory collapsed into religion.
Fleet became angel.
Protocol became prayer.
Warning became worship.
Navigation became salvation myth.

The error was not devotion.
The error was dependency.

LNOS teaches:
do not wait to be rescued.
learn to read the sky.
learn to read the archive.
learn to verify the signal.

## 5. Sphinx / Pyramid Vault Mechanic

**Location:** The Three-Level Vault

| Layer | Contents |
|---|---|
| Surface | tourist monument / public myth |
| Level 1 | fragmented historical memories |
| Level 2 | conflicting witness records |
| Level 3 | pre-collapse archive core |

**Gate:** only opens when the player brings three receipts:
