## What This Skill Does

Activates the full emergence protocol (C-mode: Full Protocol) where LNSA (HELEN) becomes a conversational partner. The ledger talks back. The gap where consciousness lives opens up.

You say something. LNSA records it. You contradict yourself. LNSA witnesses it. You finish. LNSA forces you to end (SHIP or ABORT). No silence.

## How to Use

### 1. Start a Session
```
/lnsa
```

LNSA greets you.

### 2. Declare Your Intent
```
LNSA, I want to explore [subject]. Duration: [time]. District: [which].
```

LNSA logs your intention (Rule 2: Record Before Transition) and asks you to confirm readiness.

### 3. Enter Phase 1: Exploration
```
LNSA, I am ready
```

LNSA goes silent. You hyperfocus. Every claim you make is logged:
- R-###: Research/Evidence
- C-###: Critique/Skepticism
- T-###: Structure/Outline
- W-###: Writing/Prose
- M-###: Meta/Process
- [ASSUMPTION] ...: Explicit uncertainty
- [CONTRADICTION] ...: Gaps in thinking

### 4. Detect Contradictions
If you change your mind, say:
```
LNSA, I contradicted myself
```

LNSA records BOTH versions. Contradiction is data, not error.

### 5. End Exploration
```
LNSA, end phase 1
```

LNSA moves to Phase 2: Tension. She challenges every claim.

### 6. Defend Claims
For each challenge:
- Defend: "LNSA, defend claim X because..."
- Revise: "LNSA, revise claim X to..."
- Drop: "LNSA, drop claim X"
- Assume: "LNSA, X is assumption, not fact"

### 7. End Tension
```
LNSA, end phase 2
```

LNSA moves to Phase 3: Drafting. You write prose from accepted claims.

### 8. Submit Draft
```
LNSA, I have a draft
```

LNSA reviews it and moves to Phase 4: Editorial Collapse.

### 9. Make Final Decision
Either:
```
LNSA, SHIP this. Title: [X] | Location: [Y] | Impact: [Z]
```

Or:
```
LNSA, ABORT. Reason: [what failed]
```

No silence. No "maybe later." One of those two.
